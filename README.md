# System-Tools
Basic system tools.

![image](images/SystemTools.png)

## Requirements
* Windows 10+
* Compatibity with x64

## Installation
[System Tools (Windows)](https://github.com/mrjokester0101/System-Tools/raw/refs/heads/main/Download/Windows/SystemTools.exe)
[System Tools (Linux)](https://github.com/mrjokester0101/System-Tools/raw/refs/heads/main/Download/Linux/system-tools)

## Features & Drawbacks
Features:
* Shows full system information 
* Displays hardware details
* Shows installed software
* Displays network information
* Monitors system health
* Little to no memory usage
* Good loading speed
* No side install, install once from here or files fm and use it

Drawbacks:
* Doesn't show all installed software and running processes
